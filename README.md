# Sklep Muzyczny

Prosty sklep internetowy dedykowany sprzedaży instrumentów muzycznych.

## Technologia

- Frontend: React
- Backend: Node.js + Express
- Baza danych: PostgreSQL

## Features

- Rejestracja i logowanie użytkowników
- Przeglądanie produktów
- Koszyk zakupów
- System płatności
- Panel admina
