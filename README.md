## 🚀 Jak Zacząć

1. **Zainstaluj zależności**: `npm install` w katalogach `frontend` i `backend`
2. **Przejrzyj wireframy**: Otwórz `docs/wireframes/`
3. **Zapoznaj się z design systemem**: Przeczytaj `docs/design-system/`
4. **Uruchom projekt**: `npm start` w folderze `frontend`
